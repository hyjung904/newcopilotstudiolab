# newcopilotstudiolab

마크다운으로 문서를 작성하면 GitHub Pages에서 사이트로 보여주는 저장소입니다.

## 구성

- `index.md`: 홈 화면
- `getting-started.md`: 시작 가이드
- `_config.yml`: 사이트 설정(테마: just-the-docs)

## 사용 방법

1. `.md` 파일 내용을 수정한다.
2. GitHub에 커밋/푸시한다.
3. GitHub Pages가 자동으로 사이트를 다시 빌드한다.
